Event Plumbing Demo

This page exercises native page events in Tobira. Try clicking the input and link, typing, pressing Enter, and using the submit button. The log should show capture + bubbling for click, input, change, submit, keydown, and keyup. Focus and blur are logged on the input itself, and the first input click also exercises a once listener.

Prevented link
The submit and link handlers call preventDefault() so the browser stays on this page and the log keeps growing.
The passive button listener still calls preventDefault(), but the browser should ignore it and keep defaultPrevented false.

Keyboard test: focus the input and press letters or Backspace to see keydown and keyup flow through the page DOM, including eventPhase in the log.
